## 模板概述

### 设计理念

本模板采用学术论文展示的标准风格，特点包括：

- ✅ **简洁专业**：白色背景，黑色文字，符合学术规范
- ✅ **信息层次清晰**：标题、作者、摘要、方法、结果分区明确
- ✅ **交互友好**：图片轮播、平滑滚动、响应式布局
- ✅ **易于维护**：基于Bulma框架，代码结构清晰
- ✅ **无障碍访问**：支持键盘导航和屏幕阅读器

### 技术栈

- **HTML5**：语义化标签
- **Bulma CSS Framework**：响应式框架
- **Font Awesome**：图标库
- **原生JavaScript**：无依赖，纯JS实现

## 快速开始

### 步骤1：复制模板

```bash
# 复制模板文件
cp paper-detail-template.html your-paper.html
```

### 步骤2：准备图片

在 `images/` 目录下放置您的论文图片：

```
images/
├── rendering1.png              # 渲染结果图1
├── rendering2.png              # 渲染结果图2（可选）
├── pipeline.png                # 方法流程图
├── comparison-table.png        # 结果对比表格
└── trajectory-comparison.png   # 轨迹对比图
```

### 步骤3：修改内容

打开 `your-paper.html`，按照[内容定制](#内容定制)章节修改信息。

### 步骤4：预览

使用本地服务器预览：

```bash
# 使用Python
python -m http.server 8000

# 或使用VS Code Live Server插件
# 右键HTML文件 -> Open with Live Server
```

访问：`http://localhost:8000/your-paper.html`

## 内容定制

### 1. 基本信息（必填）

#### 页面标题和描述

```html
<title>RGB-Only Gaussian Splatting SLAM - 3DAgentWorld Lab</title>
<meta name="description" content="论文摘要简短描述（150字以内）">
```

#### 论文标题

```html
<h1 class="title is-1 publication-title">
    您的论文标题
</h1>
```

#### 会议/期刊信息

```html
<h3 class="title is-3 conference-name">ICRA 2025</h3>
```

**常见会议/期刊示例：**
- `CVPR 2024` - 计算机视觉顶会
- `NeurIPS 2024` - 机器学习顶会
- `ICRA 2025` - 机器人顶会
- `IEEE TRO` - 机器人期刊
- `arXiv Preprint` - 预印本

#### 作者信息

```html
<h3 class="title is-5 authors-list">
    Author1<sup>*</sup>, 
    Author2<sup>*</sup>, 
    <a href="https://example.com" class="author-link">Hao Wang</a><sup>✉</sup>
</h3>
```

**标注说明：**
- `<sup>*</sup>` - 共同一作
- `<sup>✉</sup>` - 通讯作者
- `<sup>†</sup>` - 同等贡献
- 使用 `<a>` 标签为作者添加个人主页链接

#### 机构信息

```html
<span class="author-block">
    The Hong Kong University of Science and Technology (GuangZhou)
</span>
```

### 2. 论文链接（必填）

#### arXiv链接

```html
<a href="https://arxiv.org/abs/2502.15633" 
   target="_blank" 
   class="button is-normal is-rounded is-dark">
    <span class="icon">
        <i class="fas fa-file-pdf" style="color: orangered;"></i>
    </span>
    <span>arXiv</span>
</a>
```

#### GitHub代码链接

```html
<a href="https://github.com/your-username/your-repo" 
   target="_blank" 
   class="button is-normal is-rounded is-dark">
    <span class="icon">
        <i class="fab fa-github"></i>
    </span>
    <span>Code</span>
</a>
```

#### 项目主页（可选）

```html
<a href="https://your-project-page.com" 
   target="_blank" 
   class="button is-normal is-rounded is-dark">
    <span class="icon">
        <i class="fas fa-globe"></i>
    </span>
    <span>Project</span>
</a>
```

**如果没有某个链接，删除对应的 `<span class="link-block">` 即可。**

### 3. Abstract摘要

```html
<div class="content has-text-justified">
    <p>
        第一段：研究背景和问题。
    </p>
    <p>
        第二段：提出的方法。
    </p>
    <p>
        第三段：实验结果。
    </p>
</div>
```

**建议：**
- 每个段落3-5句话
- 使用 `<p>` 标签分段
- 保持学术语言的严谨性

### 4. 图片轮播区域

#### 添加轮播图片

```html
<div class="carousel">
    <div class="carousel-item is-active">
        <figure class="image">
            <img src="./images/rendering1.png" alt="描述图片1">
        </figure>
    </div>
    <div class="carousel-item">
        <figure class="image">
            <img src="./images/rendering2.png" alt="描述图片2">
        </figure>
    </div>
    <!-- 添加更多图片 -->
</div>
```

**注意：**
- 第一张图片必须有 `is-active` 类
- `alt` 属性用于无障碍访问，必须填写
- 图片路径相对于HTML文件

#### 修改轮播说明文字

```html
<div class="description-text">
    <p>
        您的图片说明文字...
    </p>
</div>
```

### 5. 方法说明区域

#### 方法流程图

```html
<h2 class="title is-3 has-text-centered">SLAM System Pipeline</h2>
<p>
    方法描述文字...
</p>
<figure class="image">
    <img src="./images/pipeline.png" alt="方法流程图">
</figure>
```

#### 结果对比图

```html
<h2 class="title is-3 has-text-centered mt-6">Comparison with Other Methods</h2>
<p>
    对比说明文字...
</p>
<figure class="image comparison-table">
    <img src="./images/comparison-table.png" alt="结果对比表">
</figure>
```

**提示：**
- `comparison-table` 类使图片宽度为60%（居中显示）
- 可根据需要删除此类，使用100%宽度

### 6. BibTeX引用（可选）

```html
<section class="section" id="BibTeX">
    <div class="container is-max-desktop content">
        <h2 class="title is-3">Citation</h2>
        <pre class="bibtex-code"><code>@inproceedings{yourkey2025,
    title={Your Paper Title},
    author={Author1 and Author2},
    booktitle={Conference Name},
    year={2025}
}</code></pre>
    </div>
</section>
```

**如不需要BibTeX，删除整个 `<section id="BibTeX">` 即可。**

## 图片资源

### 图片规格建议

| 用途 | 建议尺寸 | 格式 | 文件大小 |
|------|---------|------|---------|
| 轮播图 | 1920x1080 或 1600x900 | PNG/JPG | < 2MB |
| 流程图 | 1200x600 | PNG | < 1MB |
| 对比表格 | 800x600 | PNG | < 500KB |
| 轨迹图 | 1200x800 | PNG | < 1MB |

### 图片优化

**推荐工具：**
- [TinyPNG](https://tinypng.com/) - 在线压缩
- [Squoosh](https://squoosh.app/) - Google工具
- Photoshop - 导出为Web格式

**优化技巧：**
- 使用PNG格式保存图表和截图（保持清晰度）
- 使用JPG格式保存照片（减小文件大小）
- 确保图片分辨率适中（不要过高）

### 图片命名规范

```
images/
```

## 高级功能

### 1. 自定义颜色

编辑 `paper-detail.css`，修改主题色：

```css
/* 修改主题色 */
.conference-name {
    color: #3273dc;  /* 改为您想要的颜色 */
}

.author-link {
    color: #3273dc;
}

.author-link:hover {
    border-bottom-color: #3273dc;
}
```

### 2. 添加视频

在适当位置插入视频：

```html
<div class="video-container">
    <video controls width="100%">
        <source src="./videos/demo.mp4" type="video/mp4">
        您的浏览器不支持视频播放。
    </video>
</div>
```

### 3. 添加更多Section

复制现有的 `<section>` 标签结构：

```html
<section class="section">
    <div class="container is-max-desktop">
        <h2 class="title is-3 has-text-centered">新的章节标题</h2>
        <div class="content has-text-justified">
            <p>内容...</p>
        </div>
    </div>
</section>
```

### 4. 轮播图自动播放

编辑 `paper-detail.js`：

```javascript
// 找到这行代码并修改
const autoPlay = true;  // 改为true启用自动播放
const autoPlayInterval = 5000;  // 5秒切换一次
```

### 5. 禁用键盘导航

如不需要键盘导航功能，删除以下代码：

```javascript
// 删除或注释掉这段代码
document.addEventListener('keydown', (e) => {
    if (e.key === 'ArrowLeft') {
        showPreviousImage();
    } else if (e.key === 'ArrowRight') {
        showNextImage();
    }
});
```

## 常见问题

### Q1: 图片不显示？

**A:** 检查以下几点：
1. 图片路径是否正确（相对于HTML文件）
2. 图片文件名是否匹配（注意大小写）
3. 是否使用本地服务器预览（而非直接打开HTML）
4. 浏览器控制台是否有404错误

### Q2: 轮播图不工作？

**A:** 确认：
1. `paper-detail.js` 是否正确引入
2. 浏览器控制台是否有JavaScript错误
3. 图片是否至少有一张有 `is-active` 类

### Q3: 样式不生效？

**A:** 检查：
1. `paper-detail.css` 路径是否正确
2. Bulma CDN是否加载成功（检查网络）
3. 浏览器是否缓存了旧样式（Ctrl+F5强制刷新）

### Q4: 移动端显示异常？

**A:** 确保：
1. 已添加 viewport meta 标签（模板已包含）
2. 图片使用相对单位（%或auto）
3. 测试不同屏幕尺寸

### Q5: 如何修改字体？

**A:** 在 `paper-detail.css` 中修改：

```css
body {
    font-family: 'Your Font', 'Noto Sans', sans-serif;
}
```

### Q6: BibTeX复制按钮不显示？

**A:** 检查：
1. `paper-detail.js` 是否引入
2. 是否有 `id="BibTeX"` 的section
3. 浏览器是否支持 `navigator.clipboard` API

## 最佳实践

### 内容编写

1. **标题要清晰**
   - 准确概括论文主要贡献
   - 避免过长（建议15个单词以内）

2. **Abstract要精炼**
   - 3-4段结构：背景→方法→实验→结论
   - 每段3-5句话
   - 突出创新点

3. **图片要清晰**
   - 分辨率足够高
   - 文字可读
   - 标注清楚

4. **描述要准确**
   - 避免夸大
   - 数据真实
   - 引用准确

### 技术规范

1. **文件组织**
   ```
   your-paper/
   ├── index.html              # 重命名为index便于访问
   ├── paper-detail.css
   ├── paper-detail.js
   └── images/
       └── ...
   ```

2. **代码格式**
   - 使用2空格缩进
   - 保持HTML结构清晰
   - 添加必要注释

3. **无障碍访问**
   - 所有图片添加alt属性
   - 按钮添加aria-label
   - 保持语义化HTML

4. **SEO优化**
   - 填写meta description
   - 使用合适的标题层级（h1→h2→h3）
   - 添加结构化数据（可选）
