# 导师照片说明

## 📸 照片需求

本目录用于存放导师/教授照片。

## 📋 需要添加的照片

根据 `data/members.json` 的配置：

1. `wang.jpg` - 王浩教授（实验室主任）

## 🎨 照片规格要求

- **格式**：JPG 或 PNG
- **尺寸**：建议 400x400 像素以上（正方形）
- **大小**：不超过 500KB
- **背景**：建议使用专业背景或学校官方照片
- **质量**：高清、专业

## 📥 如何添加照片

1. 从学校官网或教授个人网站获取正式照片
2. 确保照片质量和专业性
3. 重命名为 `wang.jpg`
4. 复制到本目录

## 📝 注意事项

- 使用官方授权的照片
- 保持专业和学术形象
- 定期更新照片
