# 🚀 快速开始 - 3DAgentWorldLab 网站部署

## 📋 准备工作

1. **安装 Git**（如果还没有）
   - Windows: 下载 https://git-scm.com/download/win
   - 安装时保持默认选项即可

2. **创建 GitHub 账号**（如果还没有）
   - 访问 https://github.com
   - 点击 "Sign up" 注册

3. **在 GitHub 创建新仓库**
   - 登录 GitHub
   - 点击右上角 `+` → `New repository`
   - 仓库名称输入：`3dagentworld-lab`（或您喜欢的名称）
   - 选择 `Public`（公开）
   - **不要**勾选 "Initialize this repository with..."
   - 点击 `Create repository`

## 📝 方法二：手动部署

如果脚本无法运行，可以手动执行以下命令：

```powershell
# 1. 进入网站目录
cd "D:\港科广\自学\游戏项目自制\小组网页\webnew\webnew"

# 2. 初始化 Git
git init

# 3. 配置用户信息（替换成您的信息）
git config user.name "您的名字"
git config user.email "您的邮箱@example.com"

# 4. 添加所有文件
git add .

# 5. 提交
git commit -m "Initial commit: 3DAgentWorldLab website"

# 6. 关联远程仓库（替换成您的 GitHub 用户名和仓库名）
git remote add origin https://github.com/您的用户名/3dagentworld-lab.git

# 7. 推送到 GitHub
git branch -M main
git push -u origin main
```

## 🌐 启用 GitHub Pages

代码推送成功后：

1. **访问仓库页面**
   - `https://github.com/您的用户名/3dagentworld-lab`

2. **进入设置**
   - 点击 `Settings`（设置）

3. **配置 Pages**
   - 左侧菜单找到 `Pages`
   - Source: 选择 `Deploy from a branch`
   - Branch: 选择 `main`，目录选择 `/ (root)`
   - 点击 `Save`

4. **等待部署**
   - 页面顶部会显示：`Your site is live at https://...`
   - 通常需要 1-3 分钟

5. **访问网站**
   - 网址：`https://您的用户名.github.io/3dagentworld-lab/`

## 🆘 常见问题

### Q1: 推送时提示需要认证

**A**: GitHub 已不支持密码，需要使用 Personal Access Token：

1. 访问：https://github.com/settings/tokens
2. 点击 `Generate new token (classic)`
3. 勾选 `repo` 权限
4. 生成后复制 Token
5. 推送时，用户名输入 GitHub 用户名，密码输入 Token

### Q2: 脚本无法执行

**A**: PowerShell 可能禁止运行脚本，需要临时允许：

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\deploy-to-github.ps1
```

### Q3: 网站显示 404

**A**: 检查：
- GitHub Pages 是否已启用
- 等待 3-5 分钟后再试
- URL 是否正确（包含用户名和仓库名）

### Q4: 成员信息不显示

**A**: 
- 按 F12 打开开发者工具查看错误
- 确认 JSON 格式正确（运行 `python validate_json.py`）
- 清除浏览器缓存后刷新
