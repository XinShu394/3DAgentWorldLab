## 🚀 快速开始（推荐）

### 方法一：一键启动（最简单）

1. **双击运行** `启动预览服务器.bat` 文件
2. 等待几秒后，会自动弹出浏览器并显示网页
3. 如果没有自动打开，手动在浏览器输入：`http://localhost:8000/index.html`

> **注意**：关闭时请按 `Ctrl+C` 停止服务器，或直接关闭命令窗口

## 🛠️ 其他测试方法（可选）

### 方法二：手动启动服务器

如果方法一不起作用，可以手动启动：

1. **打开命令提示符**（Win+R，输入 `cmd`）
2. **进入网页文件夹**：
   ```
   cd "网页文件夹的完整路径"
   ```
3. **启动服务器**：
   ```
   python -m http.server 8000
   ```
4. **打开浏览器**，访问：`http://localhost:8000/index.html`

---

### 方法三：使用其他工具

如果您的电脑没有安装 Python，可以：

1. **安装 Live Server 插件**（如果使用 VS Code）
2. **使用其他本地服务器工具**（如 XAMPP、WAMP 等）
